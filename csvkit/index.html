    Attenzione

    La sezione per csvkit non è ancora pronta. È possibile fare riferimento al sito ufficiale.

    È un toolkit dedicato alla lettura, interrogazione, analisi, trasformazione di file CSV, scritto in Python.
    Ha fatto un po' la storia ed è un riferimento.

    Mette a disposizione diversi comandi.
