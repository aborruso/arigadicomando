# a riga di comando

Documentazione in italiano su strumenti **CLI (command-line interface)** per lavorare con dati strutturati e AI: CSV, JSON, SQLite, DuckDB, Miller, Frictionless, GDAL/OGR, Nushell e tool LLM.

Sito ufficiale: <https://arigadicomando.it/>

## Sviluppo locale

Dipendenze principali:

```bash
pip3 install mkdocs-material markdown mkdocstrings mkdocs-rss-plugin mkdocs-click mkdocs-table-reader-plugin pillow cairosvg
```

```bash
mkdocs serve
mkdocs gh-deploy --force
```
